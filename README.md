# LeetCode Solutions

A collection of LeetCode problem solutions organized by data structure and algorithm type.

## Project Structure

- `src/` - Solution implementations
  - `array/` - Array-based problems
  - `linkedlist/` - Linked list problems
  - `string/` - String manipulation problems
  - `stack/` - Stack problems
  - `queue/` - Queue problems
  - `binarytree/` - Binary tree problems
- `tests/` - Unit tests for solutions
- `notes/` - Problem notes and explanations

## Running Tests

```bash
pytest
```

## Requirements

- Python >= 3.12
